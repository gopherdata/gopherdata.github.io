Workshop: Sustainable machine learning pipelines with Pachyderm

In this hands on workshop, we will deploy the training and inference of a machine learning model into a distributed, Docker-ized pipeline with Pachyderm. The machine learning pipeline will be full reproducible and have complete provenance for results. That is, we will be able to track exactly which models produced which results and which training data sets were used to create which models, etc. Moreover, we will demonstrate how to update production models and automatically keep data pipelines in sync with data/code changes.
